Hardware events humein batate hain ki user Physical devices (Keyboard, Mouse) ke saath kaise behave kar raha hai.

1. Keyboard Events (Button Dabana)

Ek button dabana sirf ek action nahi, balki teen hisson mein hota hai:

  • onkeydown: Jaise hi button niche gaya. Ye Shift, Ctrl, Arrows sab ko track karta hai. (Gaming ke liye best).
  • onkeyup: Jab ungli utha li jaye. Ye password strength check karne ke liye best hai.
<input type="text" onkeyup="alert('Aapne key chhod di!')">

2. Mouse Movement & Tracking

Mouse events sirf click tak seemit nahi hain, ye cursor ki har harkat ko track karte hain.

onmouseover: Jab cursor element ke Andar aaye.
onmouseout: Jab cursor element se Bahar nikal jaye.
onmousedown: Click shuru hona (Dabana).
onmouseup: Click khatam hona (Chhodna).
