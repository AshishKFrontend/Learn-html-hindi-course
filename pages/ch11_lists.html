üìö Chapter 11: HTML Lists

Lists ko behtar tarike se samajhne ke liye humein unke attributes aur nesting ke "logic" ko dekhna hoga. HTML mein 3 tarah ki lists hoti hain.

1. Ordered List (<ol>) - The Logic of Sequence

Jab step-by-step jankari deni ho (jaise Recipes, Ranking), tab hum <ol> ka use karte hain.

Attributes:

  • type="1": Default Numbers (1, 2, 3...)
  • type="A": Uppercase Letters (A, B, C...)
  • type="i": Roman Numbers (i, ii, iii...)
  • start="5": Ginti kahan se shuru karni hai.
  • reversed: Ginti ulti chalegi (10, 9, 8...).
<ol type="1" reversed start="3"> <li>JavaScript</li> <li>Python</li> <li>C++</li> </ol>

2. Unordered List (<ul>) - The Concept of Bullets

Jab items ka order badalne se koi fark na pade (jaise Shopping List), tab hum <ul> use karte hain.

  • Default: Disc (Bhara hua gola ‚ö´)
  • type="circle": Khokhla gola (‚ö™)
  • type="square": Chaukor dibba (‚ñ†)

3. Definition List (<dl>) - The Glossary Style

Ye list thodi alag hai. Isme <li> nahi hota.

  • <dt>: Definition Term (Heading/Word)
  • <dd>: Definition Description (Us word ki detail)
<dl> <dt>Frontend</dt> <dd>Website ka wo hissa jo user ko dikhta hai.</dd> </dl>

4. Nested Lists (List ke andar List)

Golden Rule: Nayi list hamesha purani list ke <li> tag ke ANDAR shuru honi chahiye, uske bahar nahi.
