üìú Chapter 26: JavaScript Integration

JavaScript website mein "Jaan" phoonkta hai. HTML structure banata hai, CSS design karti hai, aur JS ye tay karta hai ki elements kaise "behave" karenge (jaise click hone par kya hoga).

1. Script Placement (Kahan Likhein?)

JavaScript ko jodne ke do main options hote hain:

  • A. In <head> section

    Script page load hone se pehle hi load ho jati hai. Isme aksar defer attribute use hota hai taaki script tabhi chale jab HTML pura load ho jaye.

  • B. At the end of <body>

    Ye Best Practice maana jata hai kyunki pehle user ko website dikh jati hai, phir piche se script load hoti hai. Isse website fast chalti hai.

2. Internal vs External JS

Internal JS

Jab <script> tag ke andar hi code likha jaye.

<script> alert('Hi!'); </script>

External JS

Jab code alag .js file mein ho.

<script src="script.js"></script>
