Phrase tags ko "Semantic Tags" bhi kaha jata hai kyunki ye sirf design nahi badalte, balki ye batate hain ki wo text kitna zaroori hai ya wo kis tarah ka data hai.

1. Semantic Tags (Importance)

Dikhne mein ye <b> aur <i> jaise hote hain, lekin inka piche ka maqsad alag hota hai.

Screen Readers: Jo log dekh nahi sakte (visually impaired), wo Screen Reader software use karte hain. <strong> tag dekhne par wo software zor dekar padhta hai.

  • <strong> (Strong Importance): Ye browser ko batata hai ki ye text bahut zaroori hai.
  • <em> (Emphasized): Iska use kisi baat par zor dene ke liye kiya jata hai (Italic look).
<p>Kripya dhyan dein: <strong>Form ki last date kal hai!</strong></p> <p>Mujhe coding <em>sach mein</em> bahut pasand hai.</p>

2. Technical Tags (Coding ke liye)

Agar aap website par programming code ya computer commands dikhana chahte hain.

Tag Kaam
<code> Code ke chhote hisse dikhane ke liye.
<kbd> Keyboard Input (Keys) batane ke liye.
<samp> Sample Output dikhane ke liye.
<var> Variable (Math/Prog) ke liye.
<p>Python mein print karne ke liye <code>print("Hello")</code> likhein.</p> <p>Restart karne ke liye <kbd>Ctrl</kbd> + <kbd>Alt</kbd> + <kbd>Del</kbd> dabayein.</p> <p>Output: <samp>Error 404: Not Found</samp></p> <p>Equation: <var>x</var> + <var>y</var> = 10</p>

3. Shortcuts: Abbr aur Acronym

<abbr> (Abbreviation): Kisi bade naam ka chhota roop. title attribute se full form dikhti hai jab mouse hover karein.

Main abhi MCA kar raha hoon. (Mouse yahan layein)

4. Special Formatting: Mark aur BDO

  • <mark>: Text ko highlight karne ke liye (Yellow marker jaisa).
  • <bdo>: Text ki disha badalne ke liye (Bi-Directional Override).
<p>Exam ke liye <mark>Chapter 6</mark> bahut zaroori hai.</p> <p><bdo dir="rtl">Ye pura sentence ulta dikhega.</bdo></p>
