üé® Chapter 5: HTML Formatting

HTML mein text ko sajaane ke liye kai tarah ke tags hote hain. Inhe hum do hisson mein baant sakte hain: Physical tags (jo sirf dikhne mein badlav karte hain) aur Semantic tags (jinka koi matlab hota hai).

1. Physical Style Tags

Ye tags text ka look turant badal dete hain.

  • <b> (Bold): Text ko mota (dark) dikhane ke liye.
  • <i> (Italic): Text ko thoda tircha dikhane ke liye.
  • <u> (Underline): Text ke niche ek line khinchne ke liye.
<p>Ye ek <b>Bold Text</b> hai.</p> <p>Ye ek <i>Italic Text</i> hai.</p> <p>Ye ek <u>Underlined Text</u> hai.</p>

2. Strike-through (Kati hui line)

Jab aapko dikhana ho ki koi cheez ab sahi nahi hai ya purani ho chuki hai, toh text ke beech mein se ek line guzaari jati hai.

Tags: <strike> aur <s> (Dono ka kaam ek hi hai - text ko "kaat" dena).

<p>Purani Price: <strike>Rs. 500</strike></p> <p>Nayi Price: Rs. 299</p>

3. Subscript aur Superscript

Ye tags math aur science ke formulas likhne ke kaam aate hain.

sub (Subscript)

Text ko normal line se thoda niche aur chhota dikhata hai.

H2O

sup (Superscript)

Text ko normal line se thoda upar aur chhota dikhata hai.

X2

4. Text Modification (Ins aur Del)

Ye tags editing ke liye use hote hain.

  • <del> (Deleted): Dikhata hai ki text delete kiya gaya hai (strikethrough jaisa).
  • <ins> (Inserted): Dikhata hai ki naya text add kiya gaya hai (underline jaisa).
<p>Mera pasandida rang <del>Laal</del> <ins>Neela</ins> hai.</p>
