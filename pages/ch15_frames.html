Frames ka use purane HTML (HTML 4.01) mein kiya jata tha taaki ek hi browser window mein kayi HTML files ko ek saath dikhaya ja sake.

‚ö†Ô∏è Warning: HTML5 mein <frameset> ko puri tarah se hatane (deprecate) ki salah di gayi hai. Aaj kal hum CSS Flexbox/Grid ya iFrames (Next Chapter) use karte hain. Ye chapter sirf knowledge ke liye hai.

1. Frameset aur Frame

Yaad rahe, jab hum <frameset> use karte hain, toh hum <body> tag ka istemal nahi karte.

  • <frameset>: Ye batata hai ki screen ko rows mein baantna hai ya columns mein.
  • <frame>: Ye batata hai ki us khaas hisse mein kaunsi HTML file dikhegi.
  • <noframes>: Agar kisi ka browser frames support nahi karta, toh ye message dikhane ke kaam aata hai.
<!-- Vertical Split (Do Columns) --> <frameset cols="30%, 70%"> <frame src="menu.html"> <frame src="content.html"> </frameset>
