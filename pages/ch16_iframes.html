üì± Chapter 16: HTML Iframes (Modern Way)

Iframe ka matlab hai Inline Frame. Ye ek "khidki" (window) ki tarah hota hai jo aapke normal page ke beech mein kahin bhi set ho sakta hai.

1. Iframe Basics

Iska sabse bada use YouTube videos ya Google Maps ko apni site par dikhane ke liye hota hai.

  • src: Us page ka link jo iframe mein dikhana hai.
  • width/height: Window ka size set karne ke liye.
  • frameborder: Border hatane ke liye "0" use karein (Waise CSS behtar hai).
<iframe src="https://www.youtube.com/embed/tgbNymZ7vqY" width="560" height="315" frameborder="0"> </iframe>

2. Security aur Sandbox Attribute

Iframes se security ka khatra ho sakta hai. Isse bachne ke liye HTML5 mein sandbox attribute laya gaya.

Sandbox Values:

  • sandbox="" : Sab kuch band (No scripts, no forms).
  • sandbox="allow-scripts" : Sirf scripts chalne do.
  • sandbox="allow-forms" : Sirf forms submit karne do.
<iframe src="site.html" sandbox="allow-scripts" width="400" height="300"></iframe>
