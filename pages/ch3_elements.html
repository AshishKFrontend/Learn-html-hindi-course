üß© Chapter 3: HTML Elements

Ek HTML Element opening tag se shuru hota hai aur closing tag par khatam hota hai. Iske beech mein "Content" hota hai.

<tagname> + Content + </tagname> = Element

üèóÔ∏è 1. Nested Elements (Tags ke andar Tags)

HTML mein hum ek element ke andar doosra element rakh sakte hain. Ise Nesting kehte hain.

Golden Rule: Jo tag sabse aakhir mein khula hai, wo sabse pehle band hoga (Last In, First Out).

<p> Ye ek paragraph hai jisme humne <b><i>Bold aur Italic</i></b> dono use kiye hain. </p> <!-- Sahi Nesting: b > i > text > /i > /b -->

üï≥Ô∏è 2. Empty Elements (Void Elements)

Kuch elements aise hote hain jinme koi content nahi hota aur isliye unka koi Closing Tag bhi nahi hota.

  • Pehchan: Inke beech mein hum kuch likh nahi sakte.
  • Examples: <br>, <hr>, <img>, <input>.
<p>Line 1</p> <br> <p>Line 2</p> <hr> <img src="photo.jpg" alt="Profile">

üß± 3. Block-level vs Inline Elements

Ye HTML ka sabse zaroori concept hai. Browser har element ko ek "Box" ki tarah dekhta hai.

A. Block-level

  • Hamesha Nayi Line se shuru hote hain.
  • Browser ki Poori Width (100%) gher lete hain.

Examples: <div>, <h1>, <p>

B. Inline

  • Nayi line se shuru nahi hote.
  • Sirf Utni hi jagah lete hain jitni zaroorat ho.

Examples: <span>, <b>, <img>
