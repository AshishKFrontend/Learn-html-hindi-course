ü§ñ Chapter 7: HTML Meta Tags

Meta tags hamesha <head> tag ke andar rakhe jate hain. Inka kaam hai "Data about data" (Metadata) dena. Ye screen par nahi dikhte, par browser aur Google ke liye bahut zaroori hain.

1. SEO Metadata (Google ke liye)

Jab aap Google par kuch search karte hain, toh jo Title aur Description dikhta hai, wo inhi tags se aata hai.

  • Description: Page ka chhota sa saar (summary).
  • Keywords: Search terms (Ab Google ise kam use karta hai).
  • Author: Website banane wale ka naam.
<meta name="description" content="Ashish ki HTML class"> <meta name="keywords" content="HTML, Coding, Hindi"> <meta name="author" content="Ashish">

2. Viewport (Mobile Friendly)

Ye tag browser ko batata hai ki website ko mobile ki screen ke hisaab se "Adjust" karein.

Zaroori: Iske bina mobile par website chhota desktop version dikhayegi (Zoom-out karke).

<meta name="viewport" content="width=device-width, initial-scale=1.0">

3. HTTP-Equiv (Auto Refresh)

Ye tags browser ko kuch "Action" lene ke liye kehte hain bina kisi programming ke.

  • Refresh: Page apne aap refresh ho jaye.
  • Redirect: User ko dusri website par bhej de.
<!-- Har 30 second mein refresh karein --> <meta http-equiv="refresh" content="30">

4. Character Set (Language Support)

Duniya bhar ki bhashayein aur symbols (emojis üòé) dikhane ke liye hum UTF-8 use karte hain.

<meta charset="UTF-8">
