üß† Chapter 24: Header Elements (The Controller)

<head> section ke andar ke elements user ko nahi dikhte, par wo browser aur search engines ko zaroori Instructions dete hain.

1. The <base> Tag (Smart Linking)

Ye tag page ke saare "Relative Links" ke liye ek Main URL set kar deta hai.

Fayda: Agar aapne <base href="https://mysite.com/assets/"> likh diya, toh aage aapko poora path likhne ki zaroorat nahi. Browser apne aap link ke aage ye address jod dega.

<!-- Page ke head mein ek baar --> <base href="https://www.mysite.com/images/" target="_blank"> <!-- Ab img tag mein sirf naam kaafi hai --> <img src="logo.png"> <!-- Browser load karega: https://www.mysite.com/images/logo.png -->

2. <link> Tag: External Connections

Iska use external resources ko HTML file se jodne ke liye hota hai.

  • CSS Connection: Stylesheet load karne ke liye.
  • Favicon: Browser tab par jo chhota sa logo dikhta hai, use lagane ke liye.
<!-- Favicon Icon --> <link rel="icon" type="image/x-icon" href="favicon.ico"> <!-- External CSS --> <link rel="stylesheet" href="style.css">

3. <style> aur <script>

Internal CSS

Page ke andar hi design likhne ke liye <style> tag use karein.

JavaScript

Logic ke liye <script> use hota hai. defer attribute script ko page load hone ke baad chalata hai.
