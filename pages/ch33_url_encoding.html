üîó Chapter 33: URL Encoding (Percent Encoding)

URL Encoding ka matlab hai: "Unsafe characters ko % sign ke saath ek code mein badal dena." URLs sirf ASCII Characters hi samajh sakte hain.

1. Kyun Zaroori Hai? (The Problem)

URLs mein spaces aur special symbols allowed nahi hote. Agar aapne file ka naam rakha My Resume.pdf, toh browser confuse ho jayega kyunki Space ka matlab hota hai "URL khatam".

Problem: Browser space ke baad wale hisse ko ignore kar deta hai, jisse link toot (break) jata hai.

2. Kaise Kaam Karta Hai? (The Solution)

Browser har unsafe character ko % aur uske Hexadecimal Code se badal deta hai.

  • Space ( ): ASCII code 32, Hex mein 20. Isliye ye ban jata hai %20.

3. Common Encoded Characters

Character Encoded Value Description
Space ( ) %20 Sabse common encoding.
Slash (/) %2F Directory separator.
Question (?) %3F Query string starter.
At (@) %40 Email address sign.

4. Encoding Functions

Programming languages (JS, Python) mein manually %20 nahi likhna padta. JS mein do main functions hain:

  • encodeURI(): Poore URL ke liye.
  • encodeURIComponent(): Kisi file name ya single value ke liye (Best).
