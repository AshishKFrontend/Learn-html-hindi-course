üé® Chapter 25: CSS Integration

HTML sirf dhancha (structure) banata hai, par use rang aur style CSS (Cascading Style Sheets) deta hai. CSS ko HTML mein jodne ke teen tarike hote hain.

1. Inline CSS

Jab aap kisi ek khaas tag ke andar hi style attribute ka use karte hain.

  • Fayda: Turant asar dikhta hai kisi specific element par.
  • Nuksan: Har tag ke liye alag se likhna padta hai, jo maintain karna mushkil hai.
<p style="color: blue; font-size: 20px;">Ye neela text hai.</p>

2. Internal CSS

Jab aap <head> section ke andar <style> tag ka use karte hain.

  • Fayda: Ek hi jagah se poore page ke elements ko control kiya ja sakta hai.
  • Nuksan: Ye sirf usi ek single page par kaam karta hai.
<head> <style> body { background-color: lightgrey; } h1 { color: red; } </style> </head>

3. External CSS (Best Practice)

Jab aap CSS ko ek alag file (e.g., style.css) mein likhte hain aur use HTML se link karte hain.

  • Fayda: Ek hi file se hazaron pages ka design badal sakte hain. Code saaf-suthra rehta hai.
<!-- HTML file mein --> <link rel="stylesheet" href="style.css"> <!-- style.css file mein --> p { color: green; }
