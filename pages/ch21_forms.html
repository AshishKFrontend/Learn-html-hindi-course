üìù Chapter 21: HTML Forms

Form ek container hota hai jisme alag-alag tarah ke input elements (text boxes, buttons, etc.) hote hain. Ye user se data lene ke kaam aata hai.

1. Form Attributes (Piche ka Logic)

Jab aap "Submit" button dabate hain, toh data kahan aur kaise jayega, ye attributes tay karte hain:

  • action: Us file ya URL ka rasta jahan data bhejna hai (e.g., login.php).
  • method: Data bhejne ka tarika:
    • GET: Data URL mein dikhta hai (Search ke liye sahi hai).
    • POST: Data chhupa kar bheja jata hai (Password ke liye zaroori).
  • enctype: Jab file/photo upload karni ho, tab multipart/form-data likhna compulsory hai.

3. Selection & Textarea

Dropdown (Select)

Jab list badi ho (City, Country).

<select> ke andar <option> lagta hai.

Textarea

Bade messages ya address likhne ke liye.

Attributes: rows (Unchai) aur cols (Chaudai).

4. Labels (Accessibility)

<label> tag ka use karna bahut zaroori hai. Label par click karne se cursor apne aap box mein chala jata hai.

<label for="user">Naam:</label> <input type="text" id="user" name="username">
