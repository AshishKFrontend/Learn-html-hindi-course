üîó Chapter 12: HTML Text Links

Text links ko Hyperlinks bhi kehte hain. Inka kaam user ko ek page se doosre page par le jana hai.

1. Anchor Tag Basics (<a>)

Hyperlink banane ke liye <a> (Anchor) tag ka use hota hai. Iska sabse zaroori attribute href (Hypertext Reference) hai.

Target Attribute: Ye batata hai ki link kahan khulega:

  • target="_self": Usi tab mein (Default).
  • target="_blank": Naye tab mein.
<a href="https://google.com" target="_blank">Google par jayein</a> <a href="about.html" target="_self">About Us</a>

2. Internal Linking (Bookmarks)

Agar aapka page bahut lamba hai aur aap chahte hain ki user ek click par page ke kisi khaas hisse (jaise Footer) par pahunch jaye, toh hum ID Selector ka use karte hain.

  • Step 1: Jis jagah jana hai wahan id="contact" lagayein.
  • Step 2: Link mein href="#contact" likhein.
<a href="#contact">Contact par Jayein</a> ... (Bahut sara content) ... <h2 id="contact">Contact Info</h2>

3. Download Links

Agar aap chahte hain ki link par click karte hi koi file (PDF, Image, Zip) download ho jaye, toh download attribute ka use hota hai.

<a href="notes.pdf" download>Download Python Notes</a>
