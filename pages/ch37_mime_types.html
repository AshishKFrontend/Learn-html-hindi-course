MIME Type ka format hamesha type/subtype hota hai. Ye browser ko batata hai ki file kis category ki hai aur use kaise handle karna hai.

MIME Type Structure

Example: text/html

  • Type (text): Category batata hai (Text, Image, Audio, etc.)
  • Subtype (html): Exact extension ya format batata hai.

Common Categories

Application Types

Ye complex data hota hai jise browser viewer ya plugin se kholta hai:

  • application/pdf: PDF document.
  • application/zip: Compressed files.
  • application/json: Data exchange format.
