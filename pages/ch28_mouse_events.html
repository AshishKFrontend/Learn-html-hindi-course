In events ka main kaam JavaScript (JS) ko trigger karna hota hai. Hum examples mein alert() ka use karenge jo browser mein ek popup message dikhata hai.

1. onclick (Single Click)

Ye sabse zyada use hone wala event hai. Jab user mouse ka Left Button ek baar dabata hai.

  • Use Case: Form submit karna, Link kholna, Button dabana.
<button onclick="alert('Button Clicked!')">Click Me</button>

2. ondblclick (Double Click)

Jab user mouse ka Left Button jaldi-jaldi do baar dabata hai.

  • Use Case: Kisi file ko open karna ya hidden secret features dikhana.
<p ondblclick="alert('You found a secret!')">Double Click Me</p>

3. oncontextmenu (Right Click)

Jab user mouse ka Right Button dabata hai. By default, ye browser ka context menu kholta hai.

üî• Pro Tip: Developers iska use Right Click Disable karne ke liye karte hain taaki koi content copy na kar sake. Iske liye return false likhna padta hai.
<div oncontextmenu="alert('Right click is disabled!'); return false;"> Right click karke dekho. </div>
