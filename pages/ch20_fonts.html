üî° Chapter 20: HTML Fonts & Typography

Halanki <font> tag purana hai aur HTML5 mein nahi use hota, par iske attributes ke concept (Face, Size, Color) aaj bhi CSS mein use hote hain.

1. Font Face (Typeface)

Aap teen tarah ke font styles define kar sakte hain:

  • Serif (Times New Roman)

    Aksharon ke kinaron par chhoti lines hoti hain.

  • Sans-Serif (Arial)

    Ekdum saaf aur seedhe akshar. Mobile screens ke liye best.

  • Monospace (Courier)

    Har akshar barabar jagah leta hai (Coding ke liye).

2. Font Size (Relative vs Absolute)

HTML fonts mein size 1 se 7 tak hota tha.

  • Absolute: size="1" (Chhota) se size="7" (Bada).
  • Relative: size="+2" (Default se 2 point bada).
<font face="Arial" size="5" color="red">Bada Text</font>
