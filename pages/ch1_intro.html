üìú 2. HTML History: Evolution

HTML waqt ke saath badalta gaya hai taaki modern features (jaise video, mobile support) ko handle kar sake.

Version Year Key Feature
HTML 1.0 1991 Sirf text aur basic links.
HTML 4.01 1999 Tables aur Frames ka zamaana.
HTML5 2014 Modern Standard. Video, Audio, aur Mobile-friendly features.

Note: Aaj ke samay mein hum HTML5 hi use karte hain.

Har HTML file ka ek fix format hota hai. Niche diye gaye editor mein code ko edit karein aur Run daba kar magic dekhein!

Components ka Purpose:

  • <html>: Ye "Root Element" hai. Poora code iske andar hota hai.
  • <head>: Isme "Metadata" hota hai (Information about the page). Ye screen par dikhta nahi hai.
  • <title>: Browser ke tab par jo naam dikhta hai.
  • <body>: Asli maal! Jo bhi aap screen par dekhte hain (Text, Images, Buttons), wo iske andar hota hai.

‚ö†Ô∏è 4. The <!DOCTYPE> Declaration

Ye koi HTML tag nahi hai, balki ek Instruction hai browser ke liye. Ye batata hai ki page HTML5 mein likha gaya hai.

Quirks Mode vs Standards Mode:

Agar aap <!DOCTYPE html> nahi likhte, toh browser "Quirks Mode" mein chala jata hai, jisse design bigad sakta hai. Likhne par ye "Standards Mode" mein chalta hai.

üè∑Ô∏è 5. Tag vs Element

Log aksar in dono ko ek hi samajhte hain, par thoda fark hai:

Tag

Sirf brackets wala hissa.

<h1> (Opening Tag)
</h1> (Closing Tag)

Element

Opening Tag + Content + Closing Tag.

<h1>Hello World</h1>
(Ye pura ek Element hai).

üî† 6. Case Sensitivity

HTML Case-Sensitive nahi hai, matlab aap <html> likho ya <HTML>, browser samajh jayega. Lekin... W3C Standard ke hisaab se hamesha Lowercase (chhote akshar) use karne chahiye.

Best Practice:

Use: <p>text</p>

Avoid: <P>TEXT</P>
