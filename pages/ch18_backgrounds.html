üé® Chapter 18: HTML Backgrounds

Purane HTML mein background ko control karne ke do main raste the. Halanki ab CSS use hoti hai, par purane code ko samajhne ke liye ye zaroori hai.

‚ö†Ô∏è Deprecated Warning: HTML5 mein bgcolor aur background attributes ka use mana hai. Ab hum CSS property background-color aur background-image use karte hain.

1. bgcolor (Solid Color)

Is attribute se kisi element ka background color set kiya jata tha.

  • Kahan lagta hai? <body>, <table>, <tr>, <td>, <th>.
  • Hex Codes: Hamesha Hex code (e.g., #FF0000) use karein kyunki ye sabse accurate hote hain.
<body bgcolor="#f0f0f0"> <table bgcolor="#ffffff"> <tr bgcolor="#ffcccc"> <td>Red Row</td> </tr> </table>

2. background (Image)

Is attribute se background mein photo lagayi jati thi.

  • Kahan lagta hai? Sirf <body> aur <table> par.
  • Tiling (Repetition): Agar image chhoti hai, toh browser use poore page par bar-bar "repeat" (tiles ki tarah) karke bhar deta hai.
<body background="pattern.jpg">
