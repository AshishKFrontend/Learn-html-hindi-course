Attributes ka kaam hai tags ko extra information dena. Ye hamesha Opening Tag ke andar likhe jate hain.

üõ†Ô∏è 1. Attribute Syntax (Likhne ka Sahi Tarika)

Attribute hamesha do hisson mein hota hai: Name aur Value.

<tagname attribute="value">Content</tagname>

Best Practice: Hamesha double quotes (" ") ka istemal karein. Agar value mein space ho (jaise class="my class"), toh quotes ke bina code crash ho jayega.

<p align="center">Ye text center mein hai.</p> <table width="100%">...</table>

üåç 2. Global Attributes (Sabse Zyada Use Hone Wale)

Global attributes wo hain jo aap kisi bhi HTML tag par laga sakte hain.

  • A. id (Unique Identity)

    Poore page mein ek ID sirf ek hi element ki ho sakti hai. Ye CSS/JS ke liye unique identifier hai.

  • B. class (Group Identity)

    Ek hi class aap bahut saare elements ko de sakte hain styling ke liye.

  • C. style (Inline CSS)

    Element ka rang, roop, aur size turant badalne ke liye.

  • D. title (Tooltip)

    Mouse hover karne par ek chhota sa info box dikhata hai.

<h1 id="main-title">Mera Blog</h1> <p class="red-text">Ye pehla laal paragraph hai.</p> <p class="red-text">Ye doosra laal paragraph hai.</p> <button style="color: blue;" title="Click Me!">Submit</button>
