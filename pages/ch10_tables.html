üìä Chapter 10: HTML Tables

HTML Table ek grid ki tarah hoti hai jisme data cells ke andar rakha jata hai (Row aur Column format mein).

1. Basic Structure (Buniyadi Dhancha)

Ek table banane ke liye humein in char main tags ki zaroorat hoti hai:

  • <table>: Main container tag jo table shuru karta hai.
  • <tr> (Table Row): Table ki ek horizontal line (row) banata hai.
  • <th> (Table Heading): Column ka naam dene ke liye. Ye text by default Bold aur Center hota hai.
  • <td> (Table Data): Row ke andar ki cells jahan asli jankari likhi jati hai.
<table border="1"> <tr> <th>Roll No</th> <th>Name</th> </tr> <tr> <td>101</td> <td>Nilima</td> </tr> </table>

2. Complex Layouts: Merging

Kabhi-kabhi humein do cells ko ek banana padta hai. Ise Merging kehte hain.

colspan

Do ya zyada columns ko ek karne ke liye (Horizontal merge).

rowspan

Do ya zyada rows ko ek karne ke liye (Vertical merge).

3. Spacing Attributes

Ye attributes table ke andar ki khali jagah (white space) ko control karte hain:

  • cellpadding: Content aur border ke beech ki doori.
  • cellspacing: Do cells (baksom) ke beech ki doori.

4. Semantic Sections

Badi tables ko manage karne ke liye hum unhe teen hisson mein baant sakte hain:

  • <thead>: Heading wala hissa.
  • <tbody>: Main data wala hissa.
  • <tfoot>: Summary ya total wala hissa (sabse niche).

5. Styling: Background & Captions

Aap table ko sundar banane ke liye in attributes ka use kar sakte hain:

  • caption: Table ka title dene ke liye (Table ke upar dikhta hai).
  • bgcolor: Background color set karne ke liye.
