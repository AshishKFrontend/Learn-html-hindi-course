ü™ü Chapter 29: Window Events

Ye attributes aamtaur par sirf <body> tag ke andar hi likhe jate hain kyunki ye poori browser window ko control karte hain.

1. onload (Page Load Hone Par)

Ye event tab fire hota hai jab poora page (HTML, CSS, Images, Scripts sab kuch) download hokar dikhne lagta hai.

  • Real Use: Welcome message dikhana ya "Loading..." spinner ko hatana.
<body onload="alert('Page 100% Load Ho Gaya!')">

2. onresize (Size Badalne Par)

Ye tab trigger hota hai jab user browser window ko mouse se pakad kar chhota ya bada karta hai.

  • Real Use: Layout adjust karna ya games mein canvas resize karna.
<body onresize="console.log('Window Size Changed')">

3. onunload (Page Band Karne Par)

Ye tab fire hota hai jab user tab band karta hai ya refresh karta hai.
