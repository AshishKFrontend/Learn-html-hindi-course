Layout ka matlab hai website ke alag-alag hisson (Header, Sidebar, Content, Footer) ko sahi jagah par set karna. Ek accha layout website ko sundar aur use karne mein aasaan banata hai.

Layout Banane Ke Tareeke

  • 1. Table-based Layout (Purana): Pehle log <table> ka use karke layout banate the, par ye SEO ke liye bura hai aur mobile par sahi nahi dikhta.
  • 2. Div-based Layout (Common): <div> aur CSS ka use karke boxes banaye jate hain. Ye flexible hota hai par isme "Semantics" (matlab) ki kami hoti hai.
  • 3. HTML5 Semantic Layout (Modern & Best): HTML5 ne naye tags diye hain jo browser ko batate hain ki page ka kaunsa hissa kya hai.

HTML5 Semantic Tags

<header>: Logo aur top links.
<nav>: Navigation links.
<section>: Content ka main part.
<aside>: Sidebar (extra info).
<footer>: Copyright/Contact info.
