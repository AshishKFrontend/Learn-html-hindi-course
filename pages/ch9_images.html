HTML mein image lagane ke liye <img> tag ka use hota hai. Yaad rahe, ye ek Empty/Void Element hai, matlab iska koi closing tag nahi hota.

1. Zaroori Attributes (Powers of Image Tag)

Image ko sahi se dikhane ke liye humein browser ko kuch jankari deni padti hai:

  • src (Source): Ye batata hai ki image kahan saved hai (URL ya file path).
  • alt (Alternative Text): Agar image load na ho, toh wahan kya likha dikhe. Ye SEO ke liye bahut zaroori hai.
  • width/height: Image ka size set karne ke liye (Pixels ya %).
<img src="photo.jpg" alt="Meri Photo" width="300" height="200">

2. Image Formats (Kaunsi image kab use karein?)

Website ki speed aur quality is baat par depend karti hai ki aap kaunsa format choose karte hain.

Format Full Form Best Use Case
JPEG Joint Photographic Experts Group Photos ke liye (High color detail).
PNG Portable Network Graphics Logos ke liye (Transparent background).
GIF Graphics Interchange Format Chhoti animations ke liye.
SVG Scalable Vector Graphics Icons ke liye (Zoom karne par nahi fat'ta).
WebP Web Picture Modern format (Size chhota, quality best).

3. Alignment (Text ke saath set karna)

Pehle images ko text ke bagal mein set karne ke liye align attribute use hota tha (Ab CSS use hoti hai, par ye seekhna zaroori hai).

  • top/middle/bottom: Vertical alignment ke liye.
  • left/right: Image ek side ho jayegi aur text uske charon taraf "wrap" (lipat) jayega.
<img src="icon.png" align="left"> <p>Ye text image ke right side mein dikhega...</p>

4. Borders (Purana Style)

border attribute se image ke charon taraf border lagayi ja sakti hai.

<img src="pic.jpg" border="5">
