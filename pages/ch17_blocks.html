üì¶ Chapter 17: HTML Blocks (Div & Span)

Inhe Generic Containers kaha jata hai. Inka apna koi visual effect nahi hota jab tak aap inpar CSS na lagayein, lekin ye structure banane ke liye sabse zaroori tags hain.

1. The Block Model (<div>)

<div> ka matlab hota hai "Division". Jab aap website ka layout banate hain, toh har bada hissa ek div hota hai.

  • Invisible Boundary: Ek div ke upar aur niche browser automatically ek line break (new line) de deta hai.
  • Nesting: Aap ek div ke andar dusra div rakh sakte hain (jise "Container" aur "Wrapper" kaha jata hai).
<div style="background: red;"> Main poori line gher raha hoon. </div>

2. The Inline Model (<span>)

<span> ek "Marker" ki tarah hai. Ye line break nahi deta.

Specific Styling: Agar aapko ek bade paragraph ke beech mein kisi ek keyword ko highlight karna hai, toh aap use span mein lapet (wrap) dete hain.

<p>Mujhe sirf <span style="color: blue;">ye word</span> highlight karna tha.</p>

3. Difference (In-Depth)

Feature <div> (Block) <span> (Inline)
Display Block Inline
New Line Hamesha nayi line se shuru hota hai. Usi line mein rehta hai.
Width 100% width leta hai. Sirf content ki width leta hai.
Usage Page Layout / Sections Text styling / Icons
