üí¨ Chapter 8: HTML Comments

Comments wo text hota hai jo aap apne code mein likhte toh hain, lekin browser use page par display nahi karta. Ye sirf aapke (developer ke) padhne ke liye hota hai.

1. Syntax (Likhne ka Tarika)

HTML mein comment likhne ka ek khaas tarika hota hai. Iske liye <!-- aur --> ka istemal kiya jata hai.

<!-- Ye ek comment hai jo screen par nahi dikhega --> <p>Ye text browser par dikhega.</p>

2. Use Cases (Kyun zaroori hain?)

Comments sirf faltu text nahi hain, inke teen bade kaam hote hain:

  • Code Documentation: Agar aap 6 mahine baad apna code dekhein, toh samajh aa jaye ki kaunsa section kis kaam ke liye hai.
  • Debugging: Agar code mein koi error hai, toh aap purane code ko delete karne ke bajaye "Comment Out" kar sakte hain taaki wo temporarily band ho jaye.
  • Collaboration: Jab team mein kaam karte hain, toh ek-doosre ke liye notes chhodne ke liye.
<!-- <p>Purana text jo ab nahi chahiye</p> --> <p>Naya aur sahi wala text yahan hai.</p>
