üåà Chapter 19: HTML Color Models

Rangon ko computer kaise samajhta hai, ye jaanna bahut zaroori hai. Computer humari tarah "Laal" ya "Neela" nahi samajhta, wo codes samajhta hai.

1. Hexadecimal System (#RRGGBB)

Ye web par sabse zyada use hone wala format hai. Ye # (Hash) se shuru hota hai.

Logic: Ye 0-9 aur A-F ke beech ki values leta hai.

  • 00 = Sabse Kam (Dark)
  • FF = Sabse Zyada (Bright)

Shortcut: Agar code repeat ho raha ho (jaise #FF0000), toh ise #F00 bhi likh sakte hain.

<h1 style="color: #FF5733;">Hex Color Title</h1> <p style="color: #000000;">Black Text</p> <p style="color: #FFFFFF;">White Text</p>

2. RGB (Red, Green, Blue)

Ye "Additive Color Model" hai. Isme teen rang milkar naya rang banate hain.

  • Range: 0 se 255 tak.
  • rgb(255, 255, 255) = Safed (White)
  • rgb(0, 0, 0) = Kaala (Black)
[Image of RGB additive color model]
<div style="background-color: rgb(0, 128, 0);">Green Box</div>

3. RGBA (Modern Extension)

Isme A ka matlab hai Alpha (Transparency/Opaqueness).

Alpha Logic:

  • 0.0 = Invisible (Poora gayab)
  • 0.5 = 50% Transparent (Dhundhla)
  • 1.0 = Solid (Bilkul saaf)
<!-- 50% Transparent Red --> <div style="background-color: rgba(255, 0, 0, 0.5);">See-through Box</div>
