üìù Chapter 30: Form Events

In events ka sequence samajhna zaroori hai: Pehle user field mein aata hai (Focus), phir kuch likhta hai (Change), phir bahar nikalta hai (Blur), aur ant mein form bhejta hai (Submit).

1. onfocus (Cursor Andar Aana)

Ye event tab fire hota hai jab user mouse se kisi input box par click karta hai ya Tab button daba kar uspar pahunchta hai.

  • Real Life Use: Input box ka background color badal dena taaki user ko pata chale ki wo abhi kahan likh raha hai (Highlighting).
<input type="text" onfocus="this.style.background='yellow'">

2. onblur (Cursor Bahar Jana)

Ye onfocus ka ulta hai. Jab user input box se bahar click karta hai ya agle box par chala jata hai.

  • Real Life Use: Email format check karna ya field khali toh nahi reh gayi, ye turant batana.
<input type="text" onblur="this.style.background='white'">

3. onchange (Value Badalna)

Ye tab fire hota hai jab user value badal kar element se bahar nikal jata hai. Dropdowns aur Checkboxes mein ye turant kaam karta hai.

4. onsubmit (The Gatekeeper)

Ye event <form> tag par lagaya jata hai. Agar ye return false karega, toh form submit nahi hoga.
